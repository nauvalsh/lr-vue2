Install Vue CLI
```
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

Create New Project
```
vue create hello-world
```

Vue DevTools
```
https://devtools.vuejs.org/guide/installation.html

```

Install Vue Router
```
vue add router
```

Install vuex
```
vue add vuex
```
